::  Open Cockpit Details  ::

Steel Battalion: Line of Contact has an impressive amount of model detail on the VTs that seems almost excessive considering that fact that the only time player would likely see it would be while watching a Replay using Free Camera. Presented below are a number of images taken during the intial VT "startup" sequence from an external view. Several of the VT models have impressively detailed cockpit interiors and starup animations.

A huge thanks to Nemesis8722 for generating all this image data.
